高频问题排行榜

后训练 / Agentic RL / RAG Agent 备战清单

后训练主线

  • 讲清 SFT、continue SFT、mixed continued training、RLVR、DPO、PPO、GRPO、DAPO 的适用边界。
  • 准备 loss、reward、KL、entropy、pass@k、benchmark、训练稳定性、reward hacking 的真实曲线口径。
  • 能手写 top-p sampling、GRPO trainer、简单 reward normalization 和 MHA。

Agentic RL 主线

  • 把 Agent 任务拆成 state、action、tool call、observation、trajectory、reward、credit assignment。
  • 准备如何用 RL 优化 tool router、function calling、长任务成功率、搜索轨迹和自我修正。
  • 比较 rule reward、model reward、process reward、outcome reward 与 human preference。

工程系统主线

  • 熟悉 verl、vLLM、Ray、rollout engine、batching、KV cache、GQA、speculative decoding。
  • 准备 RAG 召回噪声、重排、fact checking、memory 遗忘曲线、MCP、LangGraph、harness。
  • 能解释训练显存、参数/梯度/优化器状态、ZeRO/FSDP/LoRA/QLoRA。
